API Project: File Metadata Microservice

What Does It Do

  • Upload any file and get back information about that file, like:
    • Filename
    • MIME File Type
    • Filesize
  • Some other additional feature that would make sense to be done on the backend, e.g. like
    • virus scanning the file with ClamAV
    • getting back some other specific info, e.g. EXIF data from a picture
    • …

Usage

  • Drag and Drop a file below or click on the file selector to select one for uploading and wait for the upload to complete.
  • Please note that there is a 10MB file limit present.
